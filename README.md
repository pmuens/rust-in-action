# Rust in Action

Code written while following the book ["Rust in Action"](https://www.manning.com/books/rust-in-action) by Tim McNamara.

## Useful Commands

```sh
nix-shell

cargo init
cargo new <name>

cargo build [--bin <binary>]
cargo clean
cargo test
cargo bench

cargo update
cargo add <name>
cargo search <name>

cargo run [--bin <binary>] [-- arg]

cargo doc [--open]
cargo fix
cargo fmt
cargo clippy
cargo check

cargo install <name>
cargo uninstall <name>

rustup doc [--open] [--no-deps]
rustup update [toolchain]
rustup toolchain list

rustdoc <name>.rs
```
